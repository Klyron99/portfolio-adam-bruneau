# Portfolio — Adam Bruneau

Portfolio personnel présentant mon parcours et mes projets en tant qu'étudiant
en Bachelor Informatique & Cybersécurité à l'ESILV (Paris — La Défense).

## Aperçu

- Section « Mon histoire » : animation au scroll pilotée par un moteur de
  particules maison (particules formant des mots-clés du parcours)
- Thème clair / sombre
- Palette de commandes (Ctrl+K)
- Fenêtre de code animée, cartes 3D, timeline interactive

## Stack technique

100 % **HTML5 / CSS3 / JavaScript vanilla** — aucun framework, aucune
librairie externe. Uniquement Google Fonts en dépendance CDN.

## Fichiers

```
index.html   Structure de la page
style.css    Mise en forme, thèmes clair/sombre, animations
script.js    Interactions : scroll storytelling, canvas, palette de commandes…
```

## Lancer en local

Ouvrir `index.html` directement dans un navigateur, ou servir le dossier
avec un serveur statique, par exemple :

```
python -m http.server 8137
```

puis ouvrir `http://localhost:8137`.
